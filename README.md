# synthbox
My Rasperry Pi Fluidsynth Box Project

Running on  a Pi Zero, with a 16x2 LCD Display and a single rotary encoder as an interface

Depends on:
- pyFluidSynth
- RPLCD
- RPi.GPIO
- pyky040
